Projects:

Snap Project 1: Spelling Bee

Description: The Spelling Bee project is an engaging word-guessing game developed using Snap. In this game, players are presented with a hexagonally-shaped puzzle composed of seven letters, with the central letter highlighted in yellow. The goal is to form as many valid words as possible using these seven letters, ensuring that each word includes the middle letter.

Snap Project 2: 2048

Description: The 2048 Game project is a recreation of the popular sliding tile puzzle game developed using Snap. The goal of the game is to combine numbered tiles on a 4x4 grid to create a tile with the number 2048. Players slide tiles in four possible directions (up, down, left, right), and tiles with the same number merge into one when they touch. The game continues until the player achieves the 2048 tile or no more moves are possible.
